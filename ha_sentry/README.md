# Home Assistant Sentry Add-on

A Home Assistant add-on that performs daily reviews of all available updates to installed add-ons and HACS integrations, identifies potential conflicts and dependency issues, and advises whether updates are safe to install.

## About

This add-on monitors your Home Assistant installation for available updates and uses AI-powered (or heuristic) analysis to detect potential conflicts, breaking changes, and compatibility issues before you install them.
